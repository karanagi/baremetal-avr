# Bare Metal AVR Playground

This repo is a collection of projects I have written to learn bare metal programming on AVR Microcontrollers. 

As of the moment all programs are written for the [ATmega 328P](<https://www.microchip.com/en-us/product/ATMEGA328P>). 

The `Makefile` contained in these projects and the default project template has been borrowed from [this project](<https://github.com/m3y54m/start-avr>).
